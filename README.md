# C439-Group-Project

An e-commerce website for grocery shopping, built as a group project for C439. This project provides a complete online grocery store experience with product browsing, search functionality, shopping cart, checkout process, and user account management.

## Features

- Product catalog with search and filtering capabilities
- Shopping cart functionality with quantity management
- Secure checkout process with form validation
- User account system with login and registration
- Responsive design using Bootstrap 5
- Contact page with information about the company

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5.3.3
- Bootstrap Icons
